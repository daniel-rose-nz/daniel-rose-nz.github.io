CSS Display

There are 3 CSS (cascading style sheets) values which determine display behaviour on a web page.

inline displays in a horizontal sequence with otherelements inline - each flowing one after the other across the page. Any height and width properties will have no effect.

This is an example of inline elements: thing 1thing 2

The CSS for inline looks like this:

            
                display: inline;
                width: 100px;
                height: 100px;
                padding: 5px;
                border: 1px solid pink;  
                background-color: black;

inline-block displays an element as an inline-level block container. Similar to inline, except you can now set height and width values.

This is an example of inline-block elements: thing 1thing 2

The CSS for inline-block looks like this:

            
                display: inline-block;
                width: 100px;
                height: 100px;
                padding: 5px;
                border: 1px solid pink;  
                background-color: black;

While inline elements appear on the same line, block elements display on a new line and takes up the full width available. So that means block elements will occupy the entire width of its parent element

This is an example of block elements: thing 1thing 2

The CSS for block looks like this:

            
                display: block;
                width: 100px;
                height: 100px;
                padding: 5px;
                border: 1px solid pink;  
                background-color: black;
